JSONPlaceholder User and Post Details
Опис проекту:
Цей проект реалізує веб-інтерфейс для відображення інформації про користувачів та їхні пости, використовуючи API JSONPlaceholder. Проект містить три сторінки: головна сторінка з переліком користувачів, сторінка з деталями користувача та його постами, а також сторінка з деталями поста та коментарями до нього.

Використані технології:
- HTML
- CSS
- JavaScript
API JSONPlaceholder
Структура проекту:
1.index.html - Головна сторінка з переліком користувачів
2.user-details.html - Сторінка з деталями користувача та його постами
3.post-details.html - Сторінка з деталями поста та коментарями до нього
Функціональність:
index.html
- Отримує масив об'єктів користувачів з endpointа https://jsonplaceholder.typicode.com/users`.
- Виводить id та name всіх користувачів. Кожен користувач має окремий блок.
- Додає кожному блоку кнопку/посилання, при кліку на яку відбувається перехід на сторінку user-details.html, котра має детальну інформацію про обраного користувача.
user-details.html
- Виводить всю інформацію про користувача, на якого клікнули на головній сторінці.
- Додає кнопку "post of current user", при кліку на яку з'являються заголовки всіх постів поточного користувача (отримуються з endpointа https://jsonplaceholder.typicode.com/users/USER_ID/posts`).
- Додає кожному посту кнопку/посилання, при кліку на яку відбувається перехід на сторінку post-details.html, котра має детальну інформацію про обраний пост.
post-details.html
- Виводить всю інформацію про пост, на який клікнули на сторінці деталей користувача.
- Виводить всі коментарі поточного поста (отримуються з endpointа https://jsonplaceholder.typicode.com/posts/POST_ID/comments`).
Як запустити проект:
- Завантажте або клонуте цей репозиторій.
- Відкрийте файл index.html у вашому браузері.
Приклад використання:
- Перейдіть на index.html, щоб побачити список користувачів.
- Клікніть на кнопку "Детальніше" біля обраного користувача, щоб перейти на сторінку user-details.html.
- На сторінці деталей користувача клікніть на кнопку "Подивитись пости", щоб побачити заголовки постів користувача.
- Клікніть на заголовок поста, щоб перейти на сторінку post-details.html та побачити детальну інформацію про пост і його коментарі.
Стилізація проекту:
- index.html: Всі блоки з користувачами розташовані по 2 в рядок. Кнопки/посилання розташовані під інформацією про користувача.
- user-details.html: Блок з інформацією про користувача розташований зверху сторінки. Кнопка розташована нижче, на 90% ширини сторінки, по центру. Блоки з короткою інформацією про пости відображаються в ряд по 5.
- post-details.html: Блок з інформацією про пост розташований зверху. Коментарі відображаються по 4 в ряд.
